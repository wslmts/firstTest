    The important thing to remember is that  run-in will work only if the box
after the run-in box is block-level. If it is not, then the run-in box itself will
be made block-level.

Run-in Elements

Another interesting block/inline hybrid is the value run-in introduced in CSS2, which has the ability to take block-level elements them an inline part of a following element. This is useful for certain effects that are quite common in print typography, where a heading as part of a paragraph of text.
